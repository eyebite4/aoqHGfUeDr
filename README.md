# Django酒店推荐管理系统

## 核心功能
- 客房类型管理
- 酒店客房信息管理
- 客房预定处理
- 入住登记
- 续订服务
- 退房流程
- 留言反馈处理
- 系统管理权限
- 个人资料维护
- 用户管理

## 适用场景
- 酒店业客房预订管理
- 提供客户入住、续订、退房等流程支持
- 便于酒店管理客房资源和预订情况
- 收集用户反馈以优化服务质量

## 技术特点
- 使用Python语言和Django框架进行开发
- 基于MySQL数据库存储数据
- 提供详尽的1.2万字文档和PPT，便于学习和使用
